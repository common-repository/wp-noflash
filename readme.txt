== Description ==

!!!Further development on this project has stopped!!!


Adds a completely customizable and intuitive slideshow based on noflash jquery plugin (with more than 100 transitions including variations), which you can use from anywhere in your wordpress site (posts,pages,theme).

== Development Blog ==

[Nikos Web Development](http://nikos-web-development.netai.net/ "Nikos Web Development")

== Installation ==

Unpack the zip file to your wordpress plugins folder, then activate from wordpress plugins admin menu. Documentation is provided in documentation admin page.
